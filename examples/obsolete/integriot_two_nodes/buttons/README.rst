Node Description
================

this node is named 'buttons' and has one sensor:
  - button1: sends messages "up" (not pressed) and "down" (button pressed)


Connected devices
-----------------

Wiring of the sensors:

- **Button**
   - S -> D4
   - GND -> GND
   - VCC -> +3V


Functionality
-------------

The button sends the messages "up" and "down".
